# Changelog

All notable changes to the cadrianmae-issues plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.0.1] - 2026-04-13

### Fixed
- Standardise author name in plugin manifest
- Add homepage and repository fields

## [1.0.0] - 2026-04-12
### Added
- /cadrianmae-issues:bug command to file bug reports as GitHub issues
- /cadrianmae-issues:feature command to file feature requests as GitHub issues
- Automatic plugin detection from working directory (plugin:<name> labels)
- Structured issue body templates with metadata footer
- Replaces the feedback plugin's local FEEDBACK.md approach
